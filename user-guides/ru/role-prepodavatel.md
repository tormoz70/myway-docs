# Роль INSTRUCTOR (преподаватель)

Вы ведёте **занятия** в расписании и **факт посещаемости**, можете предлагать **тарифные планы** по закреплённым за вами предметам. **Удалять** записи расписания и управлять организацией в полном объёме вы **не можете**.

## Меню и ограничения

- **Боковое меню** совпадает с владельцем/админом (кроме скрытого **«Финансы»**). Не путайте «пункт виден» с «действие разрешено».
- В модалке расписания **«Удалить»** приведёт к отказу сервера — удаление только у OWNER/ADMIN.
- На **«Пропусках»** нет вкладки **«Станция / QR»**; в **«Активных пропусках»** — только свои пропуска.
- **«Финансы»**, **«Экспорт 1С»**, выдача чужих пропусков, шаблоны организации — не ваш контур (ошибка API при попытке).
