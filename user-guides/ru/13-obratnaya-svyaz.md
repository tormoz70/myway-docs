# Обратная связь

Страница **«Обратная связь»** отправляет тикеты команде разработки платформы и показывает историю ваших обращений.

## Форма нового обращения

Поля:

| Поле | Описание |
|------|----------|
| **Категория** | «Общий вопрос», «Ошибка», «Предложение», «Оплата/подписка» |
| **Тема** | Краткий заголовок, до 255 символов |
| **Сообщение** | Подробности, многострочное поле |
| **Контактный email** | Необязательно; по умолчанию может подставиться email учётной записи |
| Чекбокс согласия на обработку ПДн | Обязателен |

Кнопки: **«Отправить»**, **«Очистить»**.

После отправки — уведомление **«Обращение отправлено команде разработки»** и запись в таблице ниже.

## Таблица «Мои обращения»

Колонки: **Тема**, **Категория**, **Статус** (теги «Открыт», «В работе», «Решен», «Закрыт»), **Создан**.
