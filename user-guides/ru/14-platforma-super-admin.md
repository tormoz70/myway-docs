# Платформа (роль SUPER_ADMIN)

Раздел `/myway/platform` доступен только пользователям с ролью **SUPER_ADMIN**. Это **не** кабинет студии: здесь управление всеми организациями-подписчиками платформы.

![Платформа SUPER_ADMIN](../assets/platform-admin.png)

## Основные возможности

По вкладкам страницы (актуальные названия смотрите в интерфейсе):

1. **Организации** — таблица тенантов: тариф, статус подписки, имя, slug, контакты владельца. Есть фильтры по статусу, плану, текстовые фильтры по имени/slug/email владельца.
2. **Подписка выбранной организации** — назначение или смена тарифного плана, просмотр текущего статуса.
3. **Счета платформы** — выставление и учёт счетов на оплату услуг платформы выбранному тенанту.
4. **Платежи** — регистрация оплат по счетам.
5. **Обратная связь** — очередь тикетов пользователей; смена статуса обработки.

## Рабочая область организации под SUPER_ADMIN

Если открыть `/myway/<slug>/manage`, боковое меню **укорочено**:

- **Главная** — вместо дашборда студии показывается **панель платформы**: сводные метрики, топ тенантов по входам, полный список тенантов с действиями **«Сообщение»**, **«Блокировать»**, **«Разблокировать»**.
- **Платформа** — переход обратно на `/myway/platform`.
- **Обратная связь** — как у обычных пользователей.

**Финансовые отчёты конкретной студии** (`/manage/finance/...`) для SUPER_ADMIN **недоступны** — это внутренние данные тенанта.

## Модальные окна на главной панели

### Сообщение владельцу

Поля **«Тема»** и **«Сообщение»**; кнопка **«Отправить»**. У владельца сообщение появляется на его **«Главной»**.

### Блокировка тенанта

Поля:

- **«Режим»**: SOFT (вход есть, запись запрещена) или HARD (вход запрещён).
- **«Причина»** — текст.

Кнопка **«Применить»**.
