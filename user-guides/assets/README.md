## Как добавить или обновить снимки

1. Запустите API и фронтенд (или используйте стенд).
2. Войдите под нужной ролью (OWNER, ADMIN и т.д.).
3. Откройте раздел из таблицы выше.
4. Сохраните PNG в эту папку с **точным именем файла**.

В Markdown руководства ссылки оформлены так:

```markdown
![Подпись к рисунку](../assets/имя-файла.png)
```

Пока файла нет в репозитории, в некоторых просмотрщиках будет отображаться «битая» картинка — это ожидаемо.

При желании добавьте локальные файлы в `.gitignore`, если скриншоты не должны попадать в git.
